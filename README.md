# Superstore Sales & Profit Analysis with Machine Learning

## 📊 Project Overview
This project analyzes retail sales data and builds a machine learning model to predict profit.

## 🔍 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering (Profit Margin)
- Machine Learning Model (Random Forest)

## 📈 Insights
- Top selling sub-categories identified
- Loss-making categories detected
- Discount impact on profit analyzed

## 🤖 Machine Learning
- Model: Random Forest Regressor
- Target: Profit
- Accuracy: ~0.97 R² Score

## 🛠 Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Google Colab

## 📂 Dataset
- SampleSuperstore.csv
